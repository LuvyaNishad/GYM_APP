# LEON Brand Guidelines

## Brand Identity
* **Name:** LEON (All caps, wide spacing).
* **Concept:** "Operational Fitness." Functional, tactical, elite.

## Color Palette ("R.P.D. Navy")
* **Background:** `#0B1015` (Void Black) - Main scaffold background.
* **Surface:** `#16202A` (Tactical Grey) - Glass container color.
* **Primary:** `#3B82F6` (Electric Blue) - Active tabs, primary buttons.
* **Success:** `#4ADE80` (Herb Green) - Completed sets, "Ready" status.
* **Alert:** `#E11D48` (Laser Red) - Cancel, "Rest Needed".
* **Text:** `#E2E8F0` (Typewriter White) - Main headers.

## UI Rules
1.  **Glass Effect:**
    * Opacity: `0.6` to `0.8`.
    * Border: `Colors.white.withOpacity(0.1)` (Thin, frost border).
    * Radius: `BorderRadius.circular(24)`.
2.  **Typography:**
    * **Headers:** `FontWeight.bold`, `letterSpacing: 2.0`.
    * **Numbers:** Monospace font.
3.  **Interaction:**
    * **Haptics:** `HapticFeedback.lightImpact()` on all taps.
    * **Sound:** Mechanical clicks for UI interaction.
